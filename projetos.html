App Help Desk

Página para abertura e consulta de chamados de problemas.

Utilizada a linguagem PHP para iniciar sessão no site, através de login e senha. Na página inicial é possível escolher entre abrir um novo chamado, o qual será cadastrado pelo usuário e posteriormente salvo em um arquivo no servidor; e a outra opção é consultar os chamados criados pelo próprio usuário ou por todos os outros, dependendo do nível de usuário que está acessando o sistema.

App Orçamento Pessoal

Aplicação para cadastrar as despesas e consulta das mesmas.

Foi desenvolvido utilizando a linguagem JavaScrpit para o cadastro das despesas do usuário. Essas depesas são salvas no localStorage do navegador do usuário, após o cadastro é possível consultar as despesas e aplicar filtros para localizar elas. Sendo possível também excluir as despesas que desejar.

Jogo Mata Mosquito

Jogo com objetivo de matar os mosquitos que aparecem na tela.

Jogo desenvolvido com o uso de JavaScript, sendo possível escolher entre 3 níveis de dificuldade. O jogo possui um cronômetro e um contador de vidas onde o jogador deverá matar os mosquitos da tela antes de acabar todas as suas vidas até o tempo acabar. Caso ganhe aparecerá uma tela de parabéns pelo vitória, ou caso seja derrotado aparecerá a tela de fim de jogo.
